# 🌱 Green Earth

Green Earth is a **tree plantation campaign website** where users can explore different plants, view details, and add plants to a cart to support a tree-planting initiative.

This project demonstrates **API integration, dynamic UI rendering, category filtering, modal details, and cart functionality using JavaScript.**

---

## 🚀 Features

- 🌿 Dynamic plant listing using API
- 📂 Category-based filtering
- 🛒 Add to cart system
- 💰 Real-time total price calculation
- 📄 Plant details modal
- ⏳ Loading spinner while fetching data
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Tailwind CSS
- DaisyUI
- JavaScript (ES6)
- REST API
- Font Awesome
